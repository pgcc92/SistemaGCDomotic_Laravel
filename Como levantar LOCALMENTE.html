1) Cómo reiniciar (Dashboard + API) paso a paso

A. Reiniciar el Dashboard (puerto 8010)

  1. En la terminal donde corre el dashboard (verás “Server running on 
    http://127.0.0.1:8010”    ):
    • Presiona Ctrl + C para detenerlo.
  2. En la carpeta del proyecto:
    • cd /Applications/MAMP/htdocs/SistemaGC_Codex_LaravelAPI
  3. Limpia caches:
    • php artisan optimize:clear
  4. Levanta de nuevo:
    • php artisan serve --host=127.0.0.1 --port=8010

B. Reiniciar la API gc-data-api (puerto 8001)

  1. En la terminal donde corre la API (verás “http://127.0.0.1:8001”):
    • Presiona Ctrl + C
  2. Entra a la API:
    • cd /Applications/MAMP/htdocs/SistemaGC_Codex_LaravelAPI/services/gc-data-api
  3. Limpia caches:
    • php artisan optimize:clear
  4. Levanta de nuevo:
    • php artisan serve --host=127.0.0.1 --port=8001
